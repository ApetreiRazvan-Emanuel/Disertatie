# Details:
Repository containing the source code for The longest induced path problem, 2024 July session.

The core package (Licenta/src/main/java/core) contains the source code for the algorithms and the GraphML parser. This is the package containing the source code of algorithms within the thesis.

The benchmark package contains a few of the classes that I have used to perform tests on the algorithms and gather data about their performance.

The old package contains all of the classes that aren't used anymore at all. This includes attempts to solve the problem that weren't successful and old variants of existing classes.

# How to run the algorithms:
The benchmark package contains three classes named TestPath, TestCycle and TestGraphMLReader providing a simple example of how to run the algorithms using the GraphMLReader to obtain the input graph.
